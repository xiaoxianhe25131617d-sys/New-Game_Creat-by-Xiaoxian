# 心灵视界 Mindscape

2D横版探索解谜游戏 | Godot 4.x | GDScript

## 快速开始

用 Godot 4.x 打开 `project.godot`，按 F5 运行。

## 操作

| 按键 | 功能 |
|------|------|
| A/D | 左右移动 |
| Space | 跳跃（可变高度：松开越早跳得越低） |
| E | 互动（对话/解谜/收集） |
| F | 特殊能力（ADHD冲刺） |
| Q | 快速切换到盲人视角 |
| R | 快速切换到抑郁视角 |
| TAB | 视角轮盘（需在记忆长椅旁） |
| ESC | 暂停菜单 |

## 视角系统（屏幕后处理Shader）

每种视角有完全不同的视觉效果：

| 视角 | 视觉效果 | F键能力 |
|------|---------|---------|
| **普通** | 温暖的正常画面 | — |
| **盲人** | 全黑 + 蓝色边缘检测轮廓 | — |
| **聋人** | 冷色去饱和 + 地面振动波纹 | — |
| **ADHD** | 高饱和 + 动态色彩循环 + 几何图案 | 冲刺 |
| **抑郁** | 灰蓝去饱和 + 慢速画面扭曲 + 暗角 | — |

## 视角切换

必须在**记忆长椅**（发光标记）旁才能切换视角。按E互动坐下，然后按TAB或选择视角按钮。

## 怪物可见性

怪物**仅在其对应视角下可见**：
- 信息噪音球 → 盲人视角
- 无声嘴巴 → 聋人视角
- 阴影 → 抑郁视角

## 游戏流程（线性推进）

1. **中央广场** → 向右
2. **湖泊灯塔** → 解锁盲人视角 → 修复声波管道 → 获得碎片1
3. **旧车站** → 解锁聋人视角 → 修复货运轨道 → 获得碎片2
4. **游乐园** → 解锁ADHD视角 → 摩天轮供电 → 获得碎片3
5. **森林** → 解锁抑郁视角 → 寻找遗失纪念物 → 获得碎片4
6. **水坝** → 双视角合作（盲人+聋人） → 恢复水轮机
7. **天文台** → 三视角合作 → 启动望远镜
8. **地下管网** → 四视角全能力 → 最后碎片
9. **回到广场** → 开启时间胶囊 → 结局

## 移动手感优化

- 土狼时间（离开平台后0.08秒仍可跳跃）
- 跳跃缓冲（落地前0.1秒按空格会记住）
- 可变跳跃高度（松开空格越早→跳得越低）
- 空中控制：280px/s（地面300px/s）

## 世界探索

- 11200×4200 连续大地图
- 多层视差滚动背景（远山/云/中景山丘/树木）
- 8个区域，结构化平台布局
- 程序化BGM（每区域独特旋律）

## 系统

- 多玩家档案系统
- 自动存档（每10秒 + 关键节点）
- 46个收集品（全部在可到达表面）
- 纪念相册 + 7件主线信物追踪
- 每完成一个谜题给予下一步方向提示

## 项目结构

```
scripts/
  main.gd            - 主控制器（菜单、HUD、对话、解谜引导）
  player.gd          - 玩家物理（移动、跳跃缓冲、特殊能力）
  world.gd           - 世界构建（地形、视差、shader后处理、NPC）
  game_data.gd       - 游戏数据常量
  npc.gd             - NPC交互
  dialogue_box.gd    - 对话框UI
  monster.gd         - 怪物（视角可见性控制）
  profile_manager.gd - 多档案管理（Autoload）
  audio_manager.gd   - 音频管理（Autoload，程序化BGM+SFX）
assets/
  characters/        - 角色SVG素材
scenes/
  Main.tscn          - 主场景
```
